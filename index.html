About the Project

This project performs Exploratory Data Analysis (EDA) on a student marks dataset. It uses features like attendance percentage, internal tests, assignment scores, daily study hours, and final exam marks to understand patterns in academic performance.

The backend is implemented in Python (Streamlit) and the frontend you are viewing now is built using HTML and CSS.

Key Features

Descriptive Statistics

View summary statistics such as mean, median, min, max and standard deviation.

Distributions

Histogram and boxplot visualizations for numeric features like marks and attendance.

Correlation Analysis

Heatmap of correlations between features to identify strong relationships.

Outlier Detection

Detect potential outliers using the IQR method for numeric columns.
